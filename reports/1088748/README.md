# Anages-stoxoi
Έχοντας ασχοληθεί το τελευταίο διάστημα με την κατασκευή ιστοσελίδων και web εφαρμογων έρχομαι σε επαφή με μια ποσοτητα δεδομένων για τα οποία πρεπει να παρω την πρωτοβουλία στην κατηγοριοποίηση και τη σωστή τους τοποθέτηση, για γρήγορη πρόσβαση από τους χρήστες. Η επαφή του μαθήματος με τις σχετικές τεχνολογίες ελπίζω πως θα μου δώσει τα κατάλληλα ερεθίσματα και να διευρύνει τον τροπο σκέψης μου στην παραγωγή καλυτερης ποιότητας προϊόντων
